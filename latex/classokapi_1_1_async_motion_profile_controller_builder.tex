\doxymbox{\hyperlink{classokapi_1_1_async_motion_profile_controller_builder_ab8f0219b48e74a8ceb72280f20f9b651}{Async\+Motion\+Profile\+Controller\+Builder}} \& okapi::\+\+Async\+Motion\+Profile\+Controller\+Builder::\+not\+Parented\+To\+Current\+Task (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}

Prevents parenting the internal tasks started by this builder to the current task, meaning they will not be deleted once the current task is deleted. This can cause runaway tasks, but is sometimes the desired behavior (e.\+x., if you want to use this builder once in {\ttfamily \doxylink{main_8h_a2df3d06bc5bced154da27fce393f991f}{autonomous}} and then again in {\ttfamily opcontrol}).

Read more about this in the \+[builders and tasks tutorial]\+ (docs/\+tutorials/\+concepts/\+builders-\/and-\/tasks.\+md).

An ongoing builder. 
\end{DoxyReturn}
